Accebility in web

When websites and web tools are properly designed and coded, people with disabilities can use them. Web accessibility depends on several components working together, including web technologies, web browsers and other "user agents", authoring tools, and websites.

How to design for inclusion?

Read more
  1. To use personas with varying abilities and disabilities.
  2. Improve visibility with careful color selection and high contrast.
  3. Make content easily understandable
  4. To try using design without a mouse.
  5. Use tools such as WAVE and Color Oracle to test design’s accessibility.

How to code for inclusion?

Read more
  1. Use headings correctly to organize the structure of your content.
  2. Give your links unique and descriptive names.
  3. Use tables for tabular data, not for layout.
  4. Ensure that all content can be accessed with the keyboard alone in a logical way.
  5. Use ARIA roles and landmarks (but only when necessary).
